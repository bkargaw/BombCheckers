Game Rules

The game is the same as regular checkers and all pieces behave like classic checker pieces with the following exceptions:
  • Bomb pieces will explode when they capture a piece; during the explosion any piece within one grid space will die.
  • Shield pieces are impervious to any explosion.

Moves

* Left mouse click to select a piece or empty square (Note:- only need to select a piece once a turn)

* Press spacebar or return to end a turn
